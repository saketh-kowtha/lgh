Getting Started

Install lazyhub and go from zero to productive in under 5 minutes.

1 · Installation

Prerequisites

  • Node.js 20+ — node --version
  • gh CLI — lazyhub will guide you through install + login if missing

Install lazyhub

# one-time global install (recommended)
npm install -g lazyhub

# or run without installing
npx lazyhub

# or from source
git clone https://github.com/saketh-kowtha/lgh && cd lgh
npm install && npm start
Tip: After global install, run lazyhub from any directory that contains a Git repo and it auto-detects the GitHub remote.

2 · First Run

1

gh detection

lazyhub checks for the gh CLI. If not found, it prints platform-specific install instructions and exits cleanly — no stack traces.

2

Login

If not already authenticated, lazyhub runs gh auth login --web. On headless systems (no browser), it prompts for a Personal Access Token instead.

3

Repo detection

Inside a git repo with a GitHub remote? lazyhub auto-detects it. Outside a repo, it shows an arrow-key picker of your recent GitHub repos.

4

The TUI opens

You're in. Navigate with j/k, switch panes with Tab, press ? any time for context-aware help.

  • Sidebar — pane navigation, configurable via ~/.config/lazyhub/config.json
  • List pane — scrollable list of items for the current pane
  • Detail pane — auto-updates as you navigate the list
  • Below 100 cols: detail panel hides, Enter opens full-screen
  • Below 80 cols: sidebar hides, Tab cycles via header tabs

4 · Common Workflows

Review a PR

Navigate to PRs → j/k to select → Enter for detail → d for diff → a to approve

Merge a PR

Select PR → m → pick merge/squash/rebase → optional commit message → Enter to confirm

Work on a branch

Navigate to Branches → Space to checkout, or PRs → c to checkout PR branch

Triage issues

Issues pane → f to toggle open/closed → l to edit labels → A to assign

Watch CI

Actions pane → Enter to open run → l for logs → R to re-run failed

Clear notifications

Notifications pane → Enter to open → m mark read → M mark all read

  • n/N — jump between comment threads
  • ]/[ — next/previous file

6 · Fuzzy Search

Press / in any list pane to open fuzzy search. Type to filter in real-time:

  • PRs — title, number, author, labels, branch name
  • Issues — title, number, author, labels, milestone
  • Branches — branch name, last commit message
  • Actions — workflow name, branch, trigger
  • Notifications — title, repo, reason

Matched characters are highlighted in bold. ↑/↓ navigate results, Enter selects, Esc cancels.

7 · Tips & Tricks

  • Press ? in any view for context-aware keybinding help
  • gg jumps to the top of any list, G jumps to the bottom
  • r force-refreshes the current pane (bypasses 30s cache)
  • y copies the URL of the selected item to clipboard
  • o opens the selected item in your browser
  • Hide panes you don't use via config.json — fewer tabs = faster navigation
  • Use f to cycle between open/closed/merged states in any list
Heads up: lazyhub requires gh auth login with at minimum repo and read:org scopes. Run gh auth status to check.
