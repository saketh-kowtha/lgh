GitHub in your terminal,
the way it should be.

A lazygit-style TUI over the gh CLI — PRs, issues, diffs, line comments, Actions logs, and more. Keyboard-driven. Zero context switching.

npm install -g lazyhub

Everything GitHub, zero context switching

⎇

Pull Requests

List, review, approve, merge with strategy picker, edit labels and reviewers.

±

Diff viewer

Syntax-highlighted unified diff with line numbers, file navigation, and inline thread rendering.

💬

Line comments

Place comments on any diff line. Reply to threads, resolve, request changes — all from the keyboard.

○

Issues

List, create, close, filter. Labels, assignees, milestones — the full workflow.

▶

GitHub Actions

Stream logs, re-run failed jobs, cancel runs, filter by step — without opening the browser.

/

Fuzzy search

Press / in any pane to instantly filter by title, number, author, label, or branch name.

●

Notifications

Browse, open, and mark notifications without a browser tab.

⚡

Smart cache

30s TTL cache on all gh API calls. Press r to force-refresh any pane.

Get started in 30 seconds

npm install -g lazyhub lazyhub
# No install — runs directly npx lazyhub
git clone https://github.com/saketh-kowtha/lazyhub cd lgh npm install npm run build node dist/lazyhub.js

Requires Node.js ≥ 20 and the GitHub CLI (gh). lazyhub will detect missing dependencies and print platform-specific install instructions.
