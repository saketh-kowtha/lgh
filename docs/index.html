lazyhub

The terminal UI for GitHub you've been waiting for.

$ brew install saketh-kowtha/tap/lazyhub

Also available via npm install -g lazyhub

🤖 AI Code Review

Press A in any diff to have Claude 3.5 analyze your changes for bugs, performance leaks, and style improvements.

⚡ Vim-Style Navigation

Home-row focused navigation with j/k and lightning-fast pane switching. Context-switching is now a thing of the past.

🔍 Fuzzy Search

Instantly filter pull requests, issues, or branch lists. Jump to specific files in massive PRs with the f key.

🎨 Elite Theming

Beautifully crafted themes like Tokyo Night and Catppuccin built-in. Easily override colors via a simple JSON config.

⌨️ Essential Keybindings

Cycle Panes Tab
Navigate List j / k
Open Detail Enter
Open Diff d
Fuzzy Search /
AI Review A
Merge PR m
Approve PR a
