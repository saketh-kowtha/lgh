The keyboard-driven terminal UI for GitHub that keeps you in your flow. Built on the gh CLI.

Get Started Keybindings
$ brew install saketh-kowtha/tap/lazyhub
# or
$ npm install -g lazyhub
# then run:
$ lazyhub
