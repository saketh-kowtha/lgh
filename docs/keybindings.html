Keybindings

Every key in every view. Press ? inside lazyhub for context-aware help.

Global — works everywhere

TabNext pane
Shift+TabPrevious pane
qBack / quit (confirms if unsaved)
?Toggle context-aware help overlay
rRefresh current pane (bypass cache)
oOpen current item in browser
/Fuzzy search within current list
j / ↓Move cursor down
k / ↑Move cursor up
ggJump to top of list
GJump to bottom of list
yCopy URL of selected item to clipboard
